## 文档

[中文文档](https://boty-design.github.io/boty-design/)

## 安装
```jsx
$ npm install @boty-design/components --save

$ yarn add @boty-design/components

```

## 使用
```jsx
import { Button } from "@boty-design/components";

const App = () => (
     <Button onClick={handlerSyncClick} loading={true}  >
        异步按钮
      </Button>
);
```
